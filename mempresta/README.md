# 📚 M’Empresta

## Sobre o projeto
O **M’Empresta** é um aplicativo criado para organizar e sistematizar o processo de empréstimo de materiais didáticos em instituições de ensino. Ele ajuda a controlar livros, tablets, projetores e outros recursos acadêmicos, reduzindo perdas e atrasos na devolução.

## Funcionalidades
- Cadastro de materiais disponíveis para empréstimo  
- Registro de empréstimos e devoluções  
- Leitura e geração de QR Codes para identificação rápida dos itens  
- Histórico detalhado de transações  
- Notificações automáticas sobre prazos de devolução  
- Sincronização segura com servidor  

## Arquitetura
- Modelo **Cliente-Servidor** com comunicação via HTTPS  
- Padrão **MVC (Model-View-Controller)** para organização do sistema  
- Banco de Dados Relacional para armazenamento de usuários, itens e históricos  

## Público-alvo
- **Gestores**: responsáveis pelo acervo  
- **Usuários (alunos/professores)**: realizam empréstimos e devoluções  

## Tecnologias utilizadas
- **Figma**: prototipação das interfaces  
- **Lucidchart**: diagramas UML  
- **Banco de Dados Relacional**
